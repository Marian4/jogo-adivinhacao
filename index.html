Bem vindo, jogador!

  1. A cada partida, um número de 0 a 100 será escolhido de forma aleatória.
  2. Você terá 3 chances de adivinhar que número é esse.
  3. Caso não consiga após três tentativas, uma nova partida irá começar com um novo número.
  4. O mesmo acontecerá se você vencer a partida.
  5. Divirta-se com esse jogo sem graça :)
